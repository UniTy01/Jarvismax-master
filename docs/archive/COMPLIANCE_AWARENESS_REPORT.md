# Compliance Awareness (BE/EU)

## NOT legal advice. Basic safety awareness only.

### Checks
- GDPR triggers: email, contact, newsletter, signup, user data, tracking
- High-risk domain detection: financial advisory, medical, legal, gambling
- Always includes: provider identity, no false claims, clear service description

### When GDPR triggered
- Recommend privacy notice
- Recommend purpose limitation
- Recommend minimal data collection
- Recommend clear opt-out
