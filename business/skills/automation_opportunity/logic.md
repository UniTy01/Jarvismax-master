# Automation Opportunity — Reasoning Structure

## Step 1: Process Inventory
- List all repeatable processes in the business context
- For each: frequency, time per execution, error rate

## Step 2: Automation Candidate Scoring
- Score each process: repetitiveness × time_saved × error_reduction × feasibility
- Filter out processes that are creative or require judgment

## Step 3: ROI Estimation
- Time saved per week/month
- Cost of current manual process
- Cost of automation implementation
- Break-even timeline

## Step 4: Tool Matching
- For each candidate: which tools can automate it?
- Prefer: n8n, Zapier, Make, Python scripts, APIs

## Step 5: Implementation Priority
- Rank by: ROI / implementation_effort
- Quick wins first (high ROI, low effort)
