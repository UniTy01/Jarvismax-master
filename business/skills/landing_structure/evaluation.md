# Landing Page Structure — Quality Criteria

## Completeness
- Page structure has at least 8 sections in logical order
- Hero section has headline, subheadline, CTA, and visual guidance
- Social proof strategy covers at least 3 placement points
- CTA strategy shows progression (high → low commitment)
- At least 3 objection handlers with placement

## Structure
- Sections follow AIDA flow (Attention → Interest → Desire → Action)
- Each section has clear purpose and content guidance
- CTAs use action verbs with benefit language

## Coherence
- Hero headline matches the product's core value
- Social proof type matches the placement context
- Objection handling appears BEFORE the blocker point
- CTA progression makes sense for the conversion goal

## Red Flags
- Missing hero section or no clear CTA
- Social proof only at the bottom
- All CTAs are identical
- No objection handling
- Sections in random order (desire before attention)
