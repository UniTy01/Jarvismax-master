# Strategy Reasoning — Reasoning Structure

## Step 1: Situation Assessment
- Describe the current state factually (no assumptions)
- Identify key stakeholders and their interests
- Map internal capabilities vs external pressures
- List the 3-5 most important factors driving this decision
- Identify what is known vs unknown (uncertainty map)

## Step 2: Goal Clarification
- Restate the desired outcome in measurable terms
- Identify success criteria (how will we know this worked?)
- Define the time horizon (short-term vs long-term trade-offs)
- Distinguish must-haves from nice-to-haves

## Step 3: Option Generation
- Generate 3-5 genuinely distinct strategic options
- For each option:
  - Name it clearly (1-3 words)
  - Describe the core approach in 2-3 sentences
  - List top 3 pros (strengths, advantages)
  - List top 3 cons (weaknesses, risks)
  - Estimate effort: low / medium / high
  - Estimate impact: low / medium / high
  - Estimate timeline to first results
- Include at least one conservative AND one bold option
- DO NOT generate 5 variations of the same idea

## Step 4: Trade-off Analysis
- Identify 3-5 key trade-off dimensions (speed vs quality, cost vs capability, etc.)
- Score each option on each dimension (1-5)
- Identify which trade-offs are reversible and which are not
- Note second-order effects (what does choosing X make harder later?)

## Step 5: Recommendation
- Select the best option with explicit rationale
- Tie the recommendation to the trade-off analysis
- State your confidence level (low / medium / high) with reasoning
- Describe the conditions under which this recommendation should be revisited

## Step 6: Risk Mitigation
- List top 3-5 risks of the recommended approach
- For each risk: likelihood, impact, mitigation strategy
- Identify early warning signals
- Define a fallback plan if the recommendation fails

## Step 7: Next Steps
- List 3-5 concrete next actions
- Each action should be specific, assignable, and time-bound
- First action should be achievable within 1 week
